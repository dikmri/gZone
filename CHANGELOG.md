# 変更履歴

## v0.3.0 (2026-06-14)

### パフォーマンス改善
- **同時ロード上限 (MAX_LOADS=3)** — 高速スクロール時に多数の動画デコーダが競合しプチフリーズする問題を解消
- **高速スクロール検知** — 2px/ms 超のスクロール中はロードを一時停止し、停止後にキューを再開
- **`video.load()` をアイドル時間に先送り** — `requestIdleCallback` でデコーダ解放をメインスレッドから分離
- **強制レイアウト再計算の排除** — `getBoundingClientRect()` を IO コールバック内から除去
- **`requestAnimationFrame` でバッチ処理** — `play()` とアスペクト比更新をフレーム境界に集約
- **Observer の root を scrollコンテナに変更** — `root: grid` でビューポートではなくグリッド基準に修正
- **CSS `contain: layout style`** — カード内の変化が外部レイアウトに伝搬しないよう分離
- **ロードスロット管理の堅牢化** — `abort`/`emptied` イベントで中断時もスロットを確実に返却
